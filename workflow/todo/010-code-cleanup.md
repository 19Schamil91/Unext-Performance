# Aufgabe 010: Code-Cleanup durchfuehren

## Ziel

Nicht mehr verwendeten Code nach dem inhaltlichen Umbau sauber entfernen.

## Kontext

Der Cleanup betrifft spaeter alte Komponenten, Datenarrays, Imports, Icons, FAQ-Inhalte, SEO-Texte, Assets und auskommentierten Code.

## Vorgehen

- Nutzung vor dem Loeschen pruefen.
- Unbenutzte Dateien und Imports entfernen.
- Tests und Build nach dem Cleanup ausfuehren.

## Akzeptanzkriterien

- Kein Code wird ohne Verwendungspruefung geloescht.
- Der Build bleibt erfolgreich.
- Veraltete Inhalte sind nicht mehr im Projekt aktiv.
